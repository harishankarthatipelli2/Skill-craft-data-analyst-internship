# 🐍 Project 2 – Exploratory Data Analysis (EDA) Using Python

## 📌 Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a retail Superstore dataset using Python. The goal was to clean and preprocess the data, explore key business metrics, identify patterns and relationships, and communicate findings through meaningful visualizations.

The project demonstrates a complete data analysis workflow, from importing raw data to generating actionable business insights.

---

## 🎯 Objective

* Understand the structure of the dataset.
* Clean and preprocess the data.
* Perform Exploratory Data Analysis (EDA).
* Visualize important business metrics.
* Discover trends, patterns, and relationships within the data.
* Generate business insights to support decision-making.

---

## 📂 Dataset

The dataset contains retail sales information, including:

* Order Date
* Ship Date
* Category
* Sub-Category
* Product Name
* Customer Segment
* Region
* State
* Sales
* Profit
* Quantity
* Discount

## 📋 Project Workflow

### 1. Data Import

* Imported the dataset into Python.
* Loaded the CSV file using Pandas.
* Displayed sample records for initial inspection.

### 2. Data Understanding

* Examined dataset dimensions.
* Reviewed column names and data types.
* Generated descriptive statistics.
* Identified numerical and categorical features.

### 3. Data Cleaning

* Checked for missing values.
* Removed duplicate records.
* Corrected data types.
* Standardized column names where necessary.
* Prepared the dataset for analysis.

### 4. Exploratory Data Analysis (EDA)

Performed analysis on:

* Sales Distribution
* Profit Distribution
* Category-wise Sales
* Sub-Category Performance
* Regional Sales Analysis
* Customer Segment Analysis
* Discount vs Profit
* Monthly Sales Trends
* Top Performing Products

### 5. Data Visualization

Created visualizations using Matplotlib and Seaborn, including:

* Bar Charts
* Line Charts
* Histograms
* Scatter Plots
* Box Plots
* Correlation Heatmaps
