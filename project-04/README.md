# 📣 Project 4 – Marketing Campaign Performance & ROI Analysis Using Python

## 📌 Overview

This project analyzes marketing campaign performance using Python to evaluate campaign effectiveness, customer engagement, and Return on Investment (ROI). The objective is to transform marketing data into meaningful business insights through data cleaning, exploratory data analysis (EDA), visualization, and performance evaluation.

The analysis helps identify high-performing campaigns, optimize marketing strategies, and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze marketing campaign performance.
* Evaluate campaign effectiveness using ROI.
* Compare marketing channels and customer segments.
* Identify relationships between spend, clicks, revenue, and ROI.
* Generate actionable business insights through visualization.

---

## 📂 Dataset

The dataset contains marketing campaign information, including:

* Campaign Name
* Marketing Channel
* Campaign Spend
* Impressions
* Clicks
* Revenue
* ROI
* Campaign Duration
* Customer Segment
* Campaign Performance
