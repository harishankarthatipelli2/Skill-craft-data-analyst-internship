# 📈 Project 1 – Superstore Sales Analysis Using Microsoft Excel

## 📌 Overview

This project focuses on analyzing a retail Superstore sales dataset using **Microsoft Excel**. The objective was to transform raw sales data into meaningful business insights by performing data cleaning, creating Pivot Tables, calculating key performance indicators (KPIs), and building visual reports.

The analysis helps identify sales performance, profit trends, top-performing products, and regional business performance to support data-driven decision-making.

---

## 🎯 Objective

* Analyze retail sales performance.
* Identify profitable products and categories.
* Compare sales across different regions and states.
* Build interactive summaries using Excel.
* Generate business insights from historical sales data.

---

## 📂 Dataset

The dataset contains transactional retail sales information, including:

* Order Date
* Category
* Sub-Category
* Product Name
* Region
* State
* Sales
* Profit
* Quantity
* Discount
