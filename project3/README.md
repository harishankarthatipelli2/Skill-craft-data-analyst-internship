# 📊 Project 3 – HR Employee Attrition Analysis Dashboard (Power BI)

## 📌 Overview

This project presents an interactive **HR Employee Attrition Dashboard** developed using **Microsoft Power BI**. The objective is to analyze workforce data, identify key factors contributing to employee attrition, and provide actionable insights that support strategic HR decision-making.

The dashboard enables users to explore attrition trends across departments, job roles, age groups, and other employee attributes through interactive visualizations.

---

## 🎯 Objectives

* Analyze employee attrition patterns.
* Identify departments and job roles with high attrition.
* Measure key HR performance indicators.
* Build an interactive dashboard for workforce analysis.
* Support data-driven HR decision-making.

---

## 📂 Dataset

The project uses the **IBM HR Employee Attrition** dataset, containing employee information such as:

* Age
* Department
* Job Role
* Gender
* Education Field
* Monthly Income
* Overtime
* Job Satisfaction
* Years at Company
* Attrition Status
